Этот скилл готовит **полный русский текст** первоисточника одной находки проекта **Radar** —
то, что показывает кнопка «Весь текст» в drill-down на сайте. Работает «лениво»: только по
запросу на конкретную находку, результат кэшируется в файл.

## Вход

Два аргумента: `slug` радара и `find_id` находки (id из `data/finds/*.json` или с сайта).

## Шаг 1. Извлечь сырой текст первоисточника

Запусти:
```bash
python3 engine/extract_text.py <slug> <find_id>
```
Скрипт вернёт JSON `{kind, text, source_url, platform, ...}`:
- `kind: article` — основной текст статьи (может содержать навигационный мусор — игнорируй меню/футер).
- `kind: youtube_subs` — субтитры видео (через yt-dlp). Это и есть транскрипция.
- `kind: youtube_description` — если субтитров нет, описание видео (через YouTube Data API).
- `kind: none` — ничего не достали (нет yt-dlp/ключа, страница блокирует). Тогда сообщи
  пользователю, что нужно для полного текста (установить `yt-dlp`, задать `YOUTUBE_API_KEY`),
  и НЕ выдумывай содержание — лучше честно не создавать файл.

## Шаг 2. Перевести на русский

Переведи извлечённый `text` на естественный русский:
- Для статьи — связный полный перевод содержания (без навигации/рекламы/футера).
- Для субтитров — оформи как читаемую транскрипцию на русском (склей реплики в абзацы по смыслу,
  убери таймкоды-артефакты), не дословно-рвано.
- Для описания видео — переведи описание, отметив, что это описание, а не транскрипция.
Сохраняй технические термины узнаваемыми (Claude Code, subagents, hooks, MCP — можно оставлять
латиницей или давать рядом оригинал).

## Шаг 3. Сохранить

Запиши результат в `radars/<slug>/data/fulltext/<find_id>.ru.md` в формате:
```
# <русский заголовок находки>

> Источник: <source_url>
> Тип: перевод статьи | транскрипция видео | перевод описания видео

<полный русский текст, абзацами; для длинного — с подзаголовками>
```
После записи сайт сразу подхватит файл при клике «Весь текст» (кэш). Перезапуск не нужен.

## Замечания

- Не переводить заново, если файл уже есть и пользователь не просил обновить.
- Объёмные статьи (>20k символов) скрипт уже обрезает — переводи присланное, отметив, если текст усечён.
- Язык целевой — всегда русский (как `radar.config.language` для claude-code).
