# Radar

Ежедневный сбор интересных находок по теме с передовых площадок (X, Reddit, Hacker News,
блоги, YouTube, GitHub), структурирование и показ на сайте. Спроектирован как
**переиспользуемый шаблон**: одна тема = один радар.

Первая тема — **Claude Code**.

## Как устроено

```
collect-finds (поиск в вебе) → находки + дайджест дня → сайт (карточки, фильтры, drill-down)
                                        ▲                          │
                              профиль интересов  ◄── рейтинги (кнопки на сайте)
```

- **Движок** (`engine/`) — общий для всех тем: схема находки + шаблон сайта.
- **Радары** (`radars/<тема>/`) — данные, запросы и обучаемый профиль конкретной темы.
- **Шаблон** (`template/_radar_template/`) — образец для новой темы.
- **Скиллы** (`.claude/skills/`) — `collect-finds`, `learn-profile`.

Подробности архитектуры и правил — в [CLAUDE.md](CLAUDE.md).

## Завести новую тему

```bash
cp -r template/_radar_template radars/<новый-slug>
# отредактировать radars/<новый-slug>/radar.config.json (slug, title, taxonomy)
# отредактировать radars/<новый-slug>/prompts/queries.md
```

## Собрать находки

В Claude Code:

```
/collect-finds <slug>      # напр. /collect-finds claude-code
```

## Запустить сайт

См. `engine/site/` (появится на Этапе 3).
