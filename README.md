# Radar

Ежедневный сбор интересных находок по теме с передовых площадок (X, Reddit, Hacker News,
блоги, YouTube, GitHub), структурирование и показ на сайте. Спроектирован как
**переиспользуемый шаблон**: одна тема = один радар.

Первая тема — **Claude Code**.

## Как устроено

```text
collect-finds (поиск в вебе) → находки + дайджест дня → сайт (карточки, фильтры, drill-down)
                                        ▲                          │
              профиль интересов ◄── ⭐ рейтинги ───────────────────┤
              доска «На обдумывание» ◄── ⚑ отложить (заметки+статус)┘
```

Две петли обратной связи: **рейтинг** учит тематику (что собирать дальше), **«отложить»** —
личная очередь на размышление с заметками, которая копится в `THINKING.md`.

- **Движок** (`engine/`) — общий для всех тем: схема находки + шаблон сайта.
- **Радары** (`radars/<тема>/`) — данные, запросы и обучаемый профиль конкретной темы.
- **Шаблон** (`template/_radar_template/`) — образец для новой темы.
- **Скиллы** (`.claude/skills/`) — `collect-finds`, `learn-profile`.

Подробности архитектуры и правил — в [CLAUDE.md](CLAUDE.md).

## Завести новую тему

```bash
cp -r template/_radar_template radars/<новый-slug>
# отредактировать radars/<новый-slug>/radar.config.json (slug, title, taxonomy)
# отредактировать radars/<новый-slug>/prompts/queries.md
```

## Собрать находки

В Claude Code:

```text
/collect-finds <slug>      # напр. /collect-finds claude-code
```

## Запустить сайт

```bash
./serve.sh            # поднимет сервер и подскажет URL
# затем открыть http://127.0.0.1:8765/engine/site/index.html
```

Подробнее — [engine/site/README.md](engine/site/README.md). Сайт: лента карточек,
фильтры по тегам/платформе, drill-down с расшифровкой, ⭐ рейтинг, ⚑ «Отложить»
и страница «На обдумывание». Рейтинги/закладки — в localStorage, с экспортом в JSON.

Реализовано сверх плана:
- Источники: WebSearch + YouTube RSS (6 каналов) + YouTube Data API (поиск по всему YouTube).
  Reddit-сборщик готов (нужен OAuth). Статус — [SOURCES.md](radars/claude-code/SOURCES.md).
- «Весь текст»: полный русский перевод статьи / транскрипция YouTube (`data/fulltext/`).
- Фильтр свежести (`freshness_days`), дата новости на карточках.
- Правило: последние 2 позиции дня — всегда практические кейсы (`case-study`).
- Данные: 2 полных дня (2026-06-14, 2026-06-16) по 15 находок.
