# Сайт Radar (статика)

Чистая статика (HTML/CSS/JS), без сборки и npm. Читает `manifest.json` и `data/*.json`
радаров из корня проекта. Рейтинги (⭐) и закладки (⚑) хранятся в браузере (localStorage)
и экспортируются в JSON, совместимый с `data/feedback/ratings.json` и `saved.json`.

## Запуск локально

Из **корня проекта** (не из этой папки — данные лежат в корне):

```bash
python3 engine/build_manifest.py          # обновить индекс дней
python3 -m http.server 8765               # поднять статический сервер из корня
```

Открыть: <http://127.0.0.1:8765/engine/site/index.html>

> Почему сервер из корня: сайт читает данные по пути `../../manifest.json` и
> `../../radars/...` (см. `<meta name="data-root" content="../../">` в `index.html`).

## Деплой (Этап 4)

При публикации сайт обычно кладут в корень хостинга рядом с `data/`. Тогда поставь
`<meta name="data-root" content="">` — пути станут от корня. Всё остальное без изменений.

## Что умеет

- **Лента**: карточки находок дня (анонс), фильтры по тегам/платформе, поиск, выбор дня.
- **Drill-down**: клик по карточке → модалка с полной расшифровкой (`details`) и ссылкой на оригинал.
- **⭐ Рейтинг**: 1–5 звёзд (повторный клик снимает). Учит тематику (через экспорт → `learn-profile`).
- **⚑ Отложить**: добавить в «На обдумывание».
- **На обдумывание**: группировка по темам, статус (новое/в работе/решено/архив), личные заметки.
- **Экспорт**: выгрузка `ratings.json` + `saved.json` для коммита в репо.
