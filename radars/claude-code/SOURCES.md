## Как добавить YouTube-канал

1. Найди `channel_id`: открой канал → исходник страницы → `"channelId":"UC..."`
   (или из URL вида `youtube.com/channel/UC...`). По `@handle` напрямую не получится — редирект на consent.
2. Добавь в `sources.json` запись `type: "youtube_rss"`, `enabled: true`, URL:
   `https://www.youtube.com/feeds/videos.xml?channel_id=UC...`

## Что дальше для полного охвата соцсетей

Reddit и X через встроенные инструменты недостижимы. Варианты (по нарастанию усилий):
- **MCP-сервер Reddit** — читает сабреддиты/поиск с правильным UA. Бесплатно, нужна установка.
- **Внешний сборщик** (Python-скрипт с User-Agent → пишет в `data/finds/`) — обходит блок WebFetch.
- **Официальные API** (X — платный, YouTube Data — ключ+квоты) — максимум охвата, больше настройки.
