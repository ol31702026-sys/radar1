## 2. Синтаксис прав Tool(param:value): блокировать субагентов по модели (напр. Agent(model:opus))

🗓 **2026-06-15** · blog

**Анонс.** Гайд по субагентам (июнь 2026): добавлен синтаксис правил прав Tool(param:value) с маской * — например Agent(model:opus), чтобы запретить Opus-субагентов. Авто-режим теперь проверяет порождение субагента классификатором ДО запуска.

**Расшифровка.** Важное уточнение по безопасности субагентов: правами теперь можно матчить параметры инструмента (Tool(param:value), * как wildcard) — например блокировать субагентов на дорогой модели. Закрыт зазор: раньше субагент мог запросить заблокированное действие без ревью, теперь его спавн оценивается классификатором до запуска. Субагент не может показать интерактивный prompt — если вызов матчит ask-правило, он считается запрещённым.

`subagents · cost-tokens · automation` · Источник: [ComputingForGeeks](https://computingforgeeks.com/claude-code-subagents-guide/)
