_Полный перевод транскрипции (видео короткое)._

Claude Code 2.1.178.

Теперь можно писать правила разрешений, которые матчат **входные параметры** инструмента, а
не только его имя. Правила прав теперь срабатывают по входным параметрам.

Напишите `Agent(model:opus)`, чтобы заблокировать именно Opus-субагента, пропуская при этом
всё остальное. Поддерживаются маски (wildcard). Тот же синтаксис работает с любым
инструментом, который принимает параметры.

В авто-режиме был зазор: субагент мог запросить заблокированное действие **до** того, как его
проверит классификатор. Этот зазор закрыт. Порождение субагента теперь оценивается
классификатором **до запуска**.

Полные релиз-ноты — по ссылке. Подписывайтесь, чтобы не пропустить новые релизы.
