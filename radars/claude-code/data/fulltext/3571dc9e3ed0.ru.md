# Claude Code One Shot (Telusko)

> Источник: https://www.youtube.com/watch?v=thcZa8BIRf8
> Тип: транскрипция видео (русский перевод субтитров, связный конспект)
> Канал: Telusko (Navin Reddy)

_Перевод ключевого содержания. Видео — длинный туториал (~290 000 символов субтитров);
ниже связное изложение сути._

## Вступление

Ведущий (Navin Reddy, «Welcome back aliens») разбирает Claude Code в контексте **агентной эры**.
Сегодня агенты применяются в разных областях, и в кодинге есть несколько способов писать код:

- **Старый способ** — печатать всё руками и собирать приложение традиционно (old school).
- **Чатбот-способ** — как с ChatGPT: даёшь problem statement, получаешь код, копируешь, вставляешь
  в IDE и надеешься, что заработает. LLM-чатботы отличные, но это всё ещё ручной copy-paste.
- **Агентный способ (Claude Code)** — агент сам читает проект, пишет и правит файлы, запускает
  команды, проверяет результат, доводя задачу до рабочего состояния.

## Идея «One Shot»

Главная мысль видео — подход «one shot»: сформулировать задачу так, чтобы Claude Code выдал
**рабочий результат за один заход**, без бесконечных итераций. Это перекликается с приёмом
Plan Mode (сначала довести план до ума, потом дать исполнить) — чем точнее и полнее постановка,
тем выше шанс «попасть с первого раза».

## Что показывается в туториале

- Чем агентный кодинг отличается от чатбота (агент работает прямо в проекте, а не возвращает текст).
- Как ставить задачу, чтобы получить рабочий результат с одного прохода.
- Практическая демонстрация Claude Code на реальной задаче.

## Вывод

Хорошее вводное видео для тех, кто переходит от «копирую код из чата» к полноценному агентному
кодингу через Claude Code. Самое смотрибельное видео дня по нашим запросам (9 000+ просмотров).

> Связный перевод сути. Полная дословная транскрипция — в субтитрах оригинала (ссылка выше).
