# Субагенты Claude Code теперь порождают своих субагентов (5 уровней)

> Источник: https://www.youtube.com/watch?v=kcDsTV6Td1w
> Тип: транскрипция видео (русский перевод субтитров)
> Канал: GUIMA AI

_Полный перевод транскрипции._

Claude Code стал немного «диким». Начиная с обновления от 10 июня, ваши агенты теперь могут
**порождать собственных агентов вглубь до пяти уровней**. Даёшь ему одну задачу — а он строит
целую команду помощников.

Это мощно, но означает, что **в твои файлы быстро вносится куда больше изменений, пока ты едва
за этим следишь**.

Хорошая новость: у Claude Code есть отмена (undo). Нажми **Escape дважды**, чтобы откатить
файлы. Но откат ловит только правки, сделанные собственными инструментами редактирования
Claude. Он **не отменит** изменения в терминале — например, удалённый файл, — и привязан к
конкретной сессии.

Так что подписывайтесь на следующий выпуск и берите запасной «спасательный круг» для
по-настоящему невосстановимых случаев.

> Примечание Radar: ключевой практический вывод — при глубокой оркестрации субагентов держите
> подстраховку (Esc-Esc откатывает только правки файлов Claude, но не операции в терминале).
