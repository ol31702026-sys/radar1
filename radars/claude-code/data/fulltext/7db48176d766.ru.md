# Непрограммист собрал веб-приложение на Claude Code за выходные

_Перевод сути статьи._

## Главное

Автор прямо заявляет: **«Я не разработчик»** — и тем не менее за **одни выходные** собрал на
Claude Code рабочее веб-приложение. Это **контент-календарь**: инструмент, который позволяет
планировать публикации по разным платформам, перетаскивать посты между днями (drag-and-drop) и
отслеживать статус публикации.

Что делает кейс примечательным: несмотря на отсутствие опыта программирования, автор довёл до
рабочего состояния **функциональное веб-приложение** — с настоящей регистрацией пользователей
(signup) и хранением данных.

## Почему это важно

Ценность не в сложности продукта, а в демонстрации: **Claude Code позволяет нетехническому
автору довести идею до работающего приложения**. Это практический ответ тем, кто сомневается,
можно ли «собрать своё» без навыков разработки.

> Примечание Radar: честная оговорка — рабочий прототип ≠ продакшен с безопасностью и
> нагрузкой (как отмечают другие кейсы в подборке). Но как старт и доказательство «я могу
> построить своё» — отличный пример практического применения.
