## Что это

Репозиторий содержит актуальный список **всех системных промптов Claude Code** и связанных с
ними подсчётов токенов — по состоянию на версию **Claude Code v2.1.178 (15 июня 2026)**.
Также включён `CHANGELOG.md` по системным промптам через **211 версий**, начиная с v2.0.14.
От команды Piebald.

**Свежее (12 июня 2026):** список значительно расширен — с 350 до **515 (+165)** промптов,
это самое полное покрытие на сегодня.

Репозиторий обновляется **в течение минут после каждого релиза** Claude Code. Чтобы получать
уведомления о новых версиях — поставьте звезду (⭐) репозиторию: на каждую новую версию
создаётся релиз на GitHub.

## Почему «системных промптов» несколько

У Claude Code не один-единственный системный промпт. Вместо этого есть:

- **Большие части, добавляемые условно** — в зависимости от окружения и различных конфигов.
- **Описания встроенных инструментов** — таких как `Write`, `Bash`, `TodoWrite` (некоторые из них довольно объёмные).
- **Отдельные системные промпты для встроенных агентов** — например `Explore` и `Plan`.
- **Множество AI-функций-утилит** — компактизация диалога, генерация `CLAUDE.md`, генерация заголовка сессии и т.д.

## Чем полезно

Это первоисточник, чтобы понять «как Claude Code думает изнутри»: как сформулированы описания
27 встроенных инструментов и системные промпты субагентов (Plan/Explore/Task). Ведётся история
по версиям с токен-каунтами каждого компонента — удобно проектировать свои скиллы и субагенты
по образцу того, как это сделано в самом инструменте.
