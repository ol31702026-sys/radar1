Claude Code 2.1.178.

Правила разрешений стали «хирургическими». Теперь можно блокировать или разрешать инструменты
**на основе их входных параметров**. Одно правило не даёт Opus вообще запускаться в качестве
субагента. Матчите любое значение по маске (wildcard) или зафиксируйте конкретную модель.

Ваши правила прав наконец-то понимают, **что инструмент на самом деле делает**.
