## Философия: начинать с малого

Минимальный сетап автора **намеренно небольшой**. Что ставить в первую очередь:
- один короткий **CLAUDE.md**;
- один **scoped `.mcp.json`** — только с теми MCP-серверами Claude Code, которыми реально пользуешься;
- один **safety-хук** (хук безопасности);
- один переиспользуемый **скилл** под повторяющуюся работу;
- **субагенты — только** когда исследование или ревью иначе засорят основной контекст.

## По слоям (минимум — когда применять)

- **MCP-серверы** — GitHub, браузер/тестирование, БД или деплой-сервер со scoped-секретами.
  Ставить, когда агенту нужны внешние инструменты или данные.
- **Хуки** — обычно это хуки Claude Code вокруг событий инструментов/сессии (форматирование,
  проверки, блокировки). Для детерминированных гарантий.
- **Скиллы** — для повторяемых рабочих процессов.
- **Субагенты** — добавлять в последнюю очередь, только под research/review.

## Вывод

Антипаттерн — тащить всё подряд. Минимальный работающий набор (короткий CLAUDE.md, scoped
.mcp.json, один хук, один скилл) покрывает большинство задач, а сложное добавляется по мере
необходимости. Ровно эта философия лежит и в основе нашего проекта Radar (скиллы под конкретные
задачи, ручной запуск разрушительных действий).
