# Полный changelog Claude Code: все релиз-ноты 2026

_Перевод сути. Это агрегированный changelog — навигация и реклама опущены._

## Что это

Полная история версий Claude Code — релиз за релизом: фичи, исправления и ломающие изменения,
от беты v0.2 до свежих обновлений 2026 года.

> Примечание: эта страница документирует историю версий через май 2026. Для самых актуальных
> обновлений запусти `claude --version` в терминале.

## Пример записей (по версиям)

**v2.1.128:**
- Случайные цвета сессии — `/color` без аргументов выбирает случайный цвет сессии.
- Счётчики MCP-инструментов — `/mcp` показывает число инструментов у подключённых серверов и
  помечает серверы, которые не подключились.

(Далее по странице — детальный список изменений по каждой версии вплоть до 2.1.17x: точечные
права субагентов, вложенные агенты, Safe Mode и т.д.)

## Чем полезно

Удобный навигатор: вместо чтения разрозненных апдейтов — единая страница со всеми версиями.
Держи в закладках, чтобы быстро сверить, что нового в конкретной версии Claude Code.
