## Неделя 24 (8–12 июня 2026)

Claude Code добавляет:
- **`/cd`** — перемещение сессии в новую директорию.
- **Вложенные субагенты** — субагенты, которые могут порождать собственных субагентов.
- **Safe Mode** — изоляция «сломанных» конфигураций для отладки.

Также: расширены fallback-модели, улучшены заголовки сессий и поиск плагинов, усилены контроль
безопасности и администрирования.

### Перемещение сессии через /cd (v2.1.169)

Новая команда `/cd` переносит текущую сессию в другую рабочую директорию **без перестроения
кэша промпта** — быстро меняешь рабочую папку, не теряя контекст сессии.

## Свежее (2.1.17x, к 15–16 июня)

- **Точечные права субагентов** — правила `Tool(param:value)`, например `Agent(model:opus)`,
  чтобы блокировать конкретную модель в субагентах (маски поддерживаются).
- **Безопаснее авто-режим** — порождение субагента оценивается классификатором до запуска
  (закрыт зазор, где субагент мог запросить заблокированное действие без ревью).
- Улучшения `/doctor`, Remote Control, `/bug`.

## Чем полезно

Releasebot — удобный агрегатор: собирает релиз-ноты Claude Code из 15 источников в одну ленту
(доступно как RSS/Email/CLI/MCP/API/Slack). Хорошо для отслеживания «что прилетело» без чтения
разрозненных каналов.
