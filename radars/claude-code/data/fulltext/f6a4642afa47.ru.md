# С 15 июня — отдельный кредитный пул за программное использование Claude Code

> Источник: https://blog.getbind.co/claude-code-pricing-changes-june-15-what-youll-actually-pay-2026/
> Тип: перевод статьи
> Дата: 8 июня 2026

_Перевод сути статьи._

## Главное

Начиная с **15 июня 2026**, программное (programmatic) использование Claude Code переходит на
**отдельный кредитный пул**, тарифицируемый по полным API-ставкам — заканчивается прежняя
модель, где usage был включён в твою подписку.

Это значит, что твои реальные траты на Claude Code теперь **напрямую привязаны к расходу
токенов**. А с учётом нового токенизатора Opus 4.8, который добавляет до **35% больше токенов
на промпт**, эффективная стоимость выше, чем кажется по «ценнику».

## Кого касается

Если ты один из **18% профессиональных разработчиков**, которые пользуются Claude Code
ежедневно (по данным JetBrains 2026 Developer Ecosystem Survey), это изменение затрагивает
бюджет твоего рабочего процесса.

## Практический вывод

Стоит **пересчитать бюджеты автоматизаций/CI** до 15 июня: то, что раньше «входило в подписку»,
теперь списывается по факту по полным ставкам, плюс рост расхода токенов из-за нового
токенизатора. Особенно актуально для тех, кто гоняет Claude Code в скриптах и пайплайнах
(в том числе для будущего ежедневного сбора в нашем проекте Radar).
